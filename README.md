# 💰 Smart Saver Pro - Financial Planner

A comprehensive full-stack web application for personal finance management. Built with a focus on modern UI/UX (Glassmorphism) and predictive financial logic.

## 🌐 Live Demo
You can view the live application here:  
👉 **[http://smartplaner.gt.tc](http://smartplaner.gt.tc)**

---

## 🚀 Key Features
- **Behavioral Analytics:** Real-time financial health score (0-100) calculated based on spending ratios.
- **Future Forecast Engine:** 12-month savings projection based on current monthly habits.
- **Cinema Academy:** Integrated video gallery for financial literacy utilizing the YouTube API.
- **Goal Tracking:** Visual progress bars for tracking saving targets (e.g., Car, Laptop).
- **Bill Management:** Reminders for upcoming fixed costs and subscription services.
- **Secure Authentication:** BCRYPT password hashing and PDO prepared statements to prevent SQL Injection.
- **Modern UI:** Responsive Glassmorphism design built with advanced CSS3.

## 🛠️ Tech Stack
- **Backend:** PHP 8.x (Vanilla)
- **Database:** MySQL (Relational)
- **Frontend:** HTML5, CSS3 (Grid/Flexbox), JavaScript (ES6+)
- **Data Visualization:** Chart.js
- **Icons:** FontAwesome
